# Interview Coach Backend

Spring Boot backend for AI-powered interview preparation platform.

## Features
- JWT Authentication
- Skill Gap Analysis (Gemini API)
- Recommendations Engine
- Profile Management

## Tech Stack
- Java 21
- Spring Boot
- PostgreSQL
- Gemini API

## Setup
1. Clone repo
2. Add env variables:
   GEMINI_API_KEY=your_key
3. Run application

## API Docs
http://localhost:8080/swagger-ui.html
